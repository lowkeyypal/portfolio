## Overview

AI Image Generator is a lightweight web application that takes a natural language description and returns a generated image using OpenAI's DALL·E 2 model. A Node.js + Express REST API handles communication with the OpenAI API and serves the result to a vanilla frontend.

## Features

- **Text-to-Image Generation** — Describe anything in plain English and get a generated image back
- **DALL·E 2 Integration** — Powered by OpenAI's image generation model
- **REST API Backend** — Node.js + Express endpoint handles all API communication
- **Minimal Frontend** — Clean HTML/CSS/JS interface with no framework overhead

## Tech Stack

| Layer | Tools |
|---|---|
| **Backend** | Node.js, Express |
| **AI** | OpenAI DALL·E 2 API |
| **Frontend** | HTML, CSS, JavaScript |
| **Config** | dotenv |
